Module core.utils

AstroNvim Utilities

This module is automatically loaded by AstroNvim on during it's initialization into global variable astronvim

This module can also be manually loaded with local astronvim = require "core.utils"

astronvim.func_or_extend(overrides, default, extend)
Main configuration engine logic for extending a default configuration table with either a function override or a table to merge into the default option

Parameters:

  • overrides the override definition, either a table or a function that takes a single parameter of the original table
  • default the default configuration table
  • extend boolean value to either extend the default or simply overwrite it if an override is provided

Returns:

    the new configuration table
astronvim.default_tbl(opts, default)
Merge extended options with a default table of options

Parameters:

  • opts the new options that should be merged with the default table
  • default the default table that you want to merge into

Returns:

    the merged table

astronvim.user_plugin_opts(module, default, extend, prefix)
User configuration entry point to override the default options of a configuration table with a user configuration file or table in the user/init.lua user settings

Parameters:

  • module the module path of the override setting
  • default the default settings that will be overridden
  • extend boolean value to either extend the default settings or overwrite them with the user settings entirely (default: true)
  • prefix a module prefix for where to search (default: user)

Returns:

    the new configuration settings with the user overrides applied
